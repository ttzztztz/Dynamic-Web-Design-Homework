# 《动态网页设计》课程考查题

1  设计一个页面，内容是显示4个不同形式的九九乘法表，
即分别以矩形的两个对角线为界限的各2个三角形；
4个乘法表文字颜色随所处季节用不同的颜色显示，
并在页面的右上角显示当时当地的日期和时间，
文字大小随单日和双日有明显区别。

2  设计一个页面测试用户对10个题目（问题内容自行设定，
回答问题只有是、否两种选择）的回答， 把测试结果
（每个题目的回答结果及总的得分）保存到一个文本文件里，
并显示用户的总得分。
